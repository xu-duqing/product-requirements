# 财智宝 - 财商教育 APP 产品需求文档

**版本**: v1.0
**日期**: 2026-03-06
**优先级**: P0

---

## 产品定位

**产品名称**: 财智宝（暂定）

**目标用户**:
- 家长端：父母
- 孩子端：6-12岁儿童

**核心价值**:
- 系统化执行财商教育
- 任务 → 收入 → 账户 → 消费/储蓄/投资
- 可视化成长轨迹

**教育理念**:
- 奖励努力，而不是结果
- 义务不奖励，额外贡献才奖励
- 从"接任务的人"变成"发起项目的人"

## 产品特色

### 1. 真实交易
- 家长真实转账到孩子账户
- 或者使用虚拟货币，定期兑现

### 2. 可视化成长
- 收入曲线图
- 任务完成率
- 账户变化趋势
- 项目成长报告

### 3. 游戏化设计
- 成就徽章
- 等级系统
- 进度追踪
- 激励机制

### 4. 家长控制
- 所有奖励需要家长审核
- 账户余额限制
- 消费规则设置
- 实时监控

### 5. 项目系统
- 孩子主动发起项目

## 核心原则

### 1. 奖励努力，而不是结果
- 奖励"可控制的"（学习时间、尝试次数）
- 不奖励"不可控制的"（最终分数、他人评价）

### 2. 义务不奖励
- 收拾房间、完成作业、帮助家人是义务
- 额外贡献才值得奖励

### 3. 从被动到主动
- 从"接任务的人"变成"发起项目的人"
- 教会完整的商业流程：融资 → 执行 → 交付 → 奖励

### 4. 真实体验
- 不是模拟，是真金白银
- 成功有回报，失败有代价

## 风险控制

### 1. 预算限制
- 单任务最高奖励：50 元
- 单项目最高预算：200 元
- 同时进行项目：最多 3 个
- 月度累计预算限制

### 2. 家长审核
- 所有任务需要家长审核
- 项目资金分批发放
- 进度实时监控

### 3. 诚实教育
- 如果虚报预算，扣减奖励
- 如果质量不达标，部分返还资金
- 培养诚信

### 4. 义务与额外
- 明确区分义务和额外贡献
- 义务不做有负面后果
- 额外做才有奖励
