## 产品定位

**产品名称**: 成长资本（Growth Capital）

**核心理念**: **孩子的能力就是资本**

**目标用户**:
- 家长端：父母（投资伙伴）
- 孩子端：6-12岁儿童（项目发起人）

**核心价值**:
- 将孩子的能力转化为可管理的"资本"
- 项目 → 价值 → 收入 → 账户 → 消费/储蓄/投资
- 可视化能力成长轨迹

**合作关系**:
- 家长是投资伙伴，不是任务分配者
- 孩子是项目发起人，不是被动执行者
- 通过合作项目，共同成长

**教育理念**:
- 奖励努力，而不是结果
- 义务不奖励，额外贡献才奖励
- 从"被动执行"到"主动发起"
- 从"完成指令"到"创造价值"

## 实施计划

### Phase 1：引导期（1个月）- **当前重点**
**目标：手把手教孩子发起第1个项目，体验完整流程**

**Week 1：项目思维培养**
- [ ] 家长介绍项目概念
- [ ] 解释"能力就是资本"
- [ ] 展示项目申请流程
- [ ] 孩子理解项目与任务的区别

**Week 2：第1个项目准备**
- [ ] 家长引导孩子发现可做的项目
- [ ] 帮助孩子选择合适的第1个项目
- [ ] 手把手教她写项目申请
- [ ] 评估预算和时间，给予反馈

**Week 3：第1个项目执行**
- [ ] 孩子开始执行项目
- [ ] 家长实时监控进度
- [ ] 定期反馈和指导
- [ ] 鼓励孩子独立思考

**Week 4：第1个项目完成**
- [ ] 孩子完成项目交付
- [ ] 家长进行价值评估
- [ ] 发放项目回报
- [ ] 复盘总结，庆祝成功

**成功标准：**
- 孩子成功发起并完成第1个项目
- 孩子理解项目流程
- 孩子体验从想法到价值的完整过程
- 家长和孩子都满意合作关系

## 核心原则

### 1. 奖励努力，而不是结果
- 奖励"可控制的"（学习时间、尝试次数）
- 不奖励"不可控制的"（最终分数、他人评价）

### 2. 义务不奖励
- 收拾房间、完成作业、帮助家人是义务
- 额外贡献才值得奖励

### 3. 合作关系，不是任务分配
- 从"上级-下级"到"合作伙伴"
- 孩子发起项目，家长审核投资
- 通过合作，共同成长

### 4. 能力就是资本
- 核心理念：孩子的能力就是她的资本
- 能力可以创造价值
- 价值可以转化为收入
- 收入可以用于投资和消费

### 5. 从被动到主动
- 从"接任务的人"变成"发起项目的人"
- 教会完整的商业流程：融资 → 执行 → 交付 → 回报
